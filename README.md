# expo-reason-starter

### Getting started

1. Install dependencies: `yarn` or `npm install`
2. Build the project in a watch mode: `yarn re:watch`
3. In a separte terminal start the expo project: `yarn start`

Have fun learning and starting with ReasonML ❤️
